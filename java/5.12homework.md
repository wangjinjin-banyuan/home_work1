1. 
```
class Outer{
 public static void main(String[] args){
  new Inner(); 
 } 
 class Inner { 
  Inner() {} 
 }  
}
```
结果：编译错误

原因：内部类不能直接创建，必须先创建外部类对象后才能创建内部类，比如：new Outer().new Inner();
